CSS rules

1. Naming convention

Classes written in lowercase. Words splitted by - (a dash).

HTML

<div class="content">
    <div class="sample-info"></div>
    <div class="additional-content"></div>
  </div>

CSS/LESS

.content {
    // ...
    .sample-info {
      // ...
    }
    .additional-content {
      // ...
    }
  }

2. Structure

Do not create classes like tab-content-active-button-dark . It's better to create five classes. Then, your code will be reusable (e.g. active can be used in another place), more readable and much more easier to maintain.

HTML

<div class="tab">
    <span class="title active">First tab</span>
    <span class="title">Second tab</span>
    <div class="content active">
      <button class="btn dark">First button</button>
    </div>
    <div class="content">
      <button class="btn">Second button</button>
    </div>
  </div>

CSS/LESS

.tab {
    .title {
      // ...
      &.active {
        text-decoration : underline;
      }
    }
    .content {
      // ...
      &.active {
        background-color: #f3f3f3;
      }
      .btn {
        // ...
        &.dark {
          background-color: #333;
        }
      }
    }
  }

3. IDs and tags

Ids... Just don't use it. If you have to, ask somebody else and find out that you don't have to.

Try to use CSS without tag names. If you write div.myclass - what happens, when somebody change div to span ?
There are some exceptions. E.g. if you know that you want to style specific tags. For instance you know that all the headers in your app should be green. Then you can write h1,h2,h3,h4,h5,h6 { color: green } . But as long as you don't want to specify the general rule - avoid to use tags in CSS.

4. Number of styles

Don't write any styles if you really don't need them. It's a simple rule but not as easy as it's seems :) Remember: styles are inherited.

Wrong

<div class="first">
    <div class="second">
      <div class="third"></div>
    </div>
  </div>

  // css file:
  .first {
    color: green;
    .second {
      color: green;
    }
  }
  .third {
    color: green;
  }

Good

<div class="first">
    <div>
      <div></div>
    </div>
  </div>

  // css file:
  .first {
    color: green;
  }

And remember: always check if a similar solution is not implemented in Twitter Bootstrap or somewhere else! I haven't use any other stylesheet than bootstrap and prettyprint in this section related with CSS styles.
